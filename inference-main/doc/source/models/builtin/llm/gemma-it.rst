========================================
gemma-it
========================================

- **Context Length:** 8192
- **Model Name:** gemma-it
- **Languages:** en
- **Abilities:** chat
- **Description:** Gemma is a family of lightweight, state-of-the-art open models from Google, built from the same research and technology used to create the Gemini models.
